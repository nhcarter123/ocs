# ocs
Open chess tournament software

### Getting started

Download [NodeJS](https://nodejs.org/en/download/), if you don't have it. (I'm using v16.5.0)

Install yarn, if you don't have it
```
npm i yarn -g
```

Install dependencies
```
yarn
```

Start the app
```
npm run start
```
